# NLP-Based Chatbot (Python)

## Overview

This project is a simple NLP-based chatbot built using Python. It uses text preprocessing and fuzzy string matching to understand user input and return appropriate responses. The project demonstrates fundamental NLP and Python development skills.

## Features

* Intent recognition using fuzzy matching
* Text preprocessing and cleaning
* JSON-based response system
* Modular and easy-to-extend code

## Technologies Used

* Python
* JSON
* fuzzywuzzy
* pypdf

## Project Structure

```
chatbot_project/
├── chatbot.py
├── responses.json
├── requirements.txt
└── README.md
```

## How to Run

Install dependencies:

```
pip install -r requirements.txt
```

Run chatbot:

```
python chatbot.py
```

## Example

```
You: hello
Bot: Hello! How can I help you?
```
